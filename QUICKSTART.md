# Quick Start Guide

## üöÄ Getting Started in 5 Minutes

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Run the Application
```bash
python ADVANCED_MODEL_WITH_SPEECH_RECOGNITION_TEXT_RECOGNITION.py
```

### Step 3: Select Mode
When the application starts, say one of these voice commands:
- **"Inside Mode"** - For indoor navigation
- **"Outside Mode"** - For outdoor navigation

### Step 4: Start Using
- The system will automatically detect and announce objects
- Use voice commands like **"Find chair"** to search for specific objects
- Press **`C`** to calibrate for better accuracy
- Press **`Q`** to quit
